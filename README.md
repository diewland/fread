# fread
Feed reader ( [live](http://diewland.github.io/fread) )

#### cursor

* J - Move Down
* K - Move Up
* . - Move Top
* SHIFT+G - Move Bottom
* / - Toggle Filter mode

#### item

* ENTER - Expand item
* L - Collapse item
 
#### other

* R - Refresh fread
* O - Open in new window
* P - Share to Google+
